# 🎌 Anime/Manga Prompt Generator Chip

Generates anime and manga style art prompts for AI image generators like Midjourney, Stable Diffusion, and DALL-E.

## Features
- Random anime/manga character generation
- Multiple art styles (shonen, shojo, seinen, chibi)
- Japanese-inspired backgrounds
- Professional anime art prompts

## Quick Start
```bash
pip install -r requirements.txt
python prompt_generator.py
